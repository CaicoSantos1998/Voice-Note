# Voice-Note
This project is a notepad app targeting users who cannot read or write. The app will feature a voice system to assist the user with note-taking and commands. The audio will be responsible for informing the user of everything that is happening in the app. Everything will depend on how the user uses the app.

## Technologies
- **Language:** Java
- **Android SDK:** 16.0 (Upside Down Cake / Android 14+)
- **Gradle:** 9.1.0
